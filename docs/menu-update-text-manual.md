# つばき亭 メニュー更新マニュアル（文字だけ版）

このマニュアルは、microCMSでホームページのメニュー金額・写真・表示/非表示を変更するための説明です。

対象ページ:

- 洋食ツバキ亭 メニュー
- ツバキ&デリ デリメニュー

## まず知っておくこと

メニュー情報はmicroCMSで管理します。

microCMSで保存した内容は、ホームページのPC表示とスマホ表示の両方に反映されます。PC用とスマホ用を別々に直す必要はありません。

初期設定が終わるまでは、microCMSで変更してもホームページには反映されません。初期設定はYoshi側で行います。

## 普段触ってよい項目

店主さん・スタッフさんが普段触るのは、基本的に次の項目だけです。

- 表示する
- 店舗
- メニュー区分
- セクション名
- 品名
- 金額
- 金額表示
- 写真

説明文も必要な時だけ触ってOKです。

「店舗」「メニュー区分」「セクション名」は、選択式の欄がある場合、必ず選択式の欄を使ってください。手入力すると、入力ミスでホームページに表示されないことがあります。

選択例:

- 店舗: ツバキ亭 / ツバキ&デリ
- メニュー区分: Lunch / Dinner / Course / デリ
- セクション名: Lunch Menu / Dinner Menu / Course / ツバキ&デリ メニュー

## 普段触らない項目

次の項目は、表示の仕組みに関わるので基本的に触らないでください。

- セクション説明
- 並び順
- 既存写真パス
- 旧・店舗
- 旧・メニュー区分
- 旧・セクション名

ここを変えると、別の店舗に表示されたり、メニューの場所が変わったり、表示されなくなることがあります。

古いテキスト入力欄として「旧・店舗」「旧・メニュー区分」「旧・セクション名」が残っている場合があります。これは過去データ用なので、新しく追加する時は触らないでください。

## 金額の入れ方

### 普通の金額

単一価格の場合は「金額」に数字だけ入力します。

例:

```text
1200
```

ホームページでは次のように表示されます。

```text
￥1,200
```

### S/Lなど複数価格がある場合

Sサイズ・Lサイズなど、複数の価格を表示したい時は「金額表示」にそのまま表示したい文字を入力します。

例:

```text
S ￥650 / L ￥750
```

「金額表示」に文字が入っている場合、ホームページでは「金額」より「金額表示」が優先されます。

### 金額表示を消したい場合

「金額表示」が残っていると、そちらが優先されます。

普通の単一価格に戻したい時は、

1. 「金額」に数字を入れる
2. 「金額表示」を空欄にする

この2つを行ってください。

## 写真の変え方

1. microCMSで変更したいメニューを開く
2. 「写真」の欄を探す
3. 新しい写真をアップロードする
4. 保存する
5. ホームページを再読み込みして確認する

写真を入れると、その写真がホームページに表示されます。

写真が空の場合は、既存の写真または空の枠が表示されます。

## 表示・非表示の切り替え

一時的にメニューを隠したい時は「表示する」をOFFにします。

削除は基本的にしないでください。

間違えて削除すると戻すのが面倒なので、使わないメニューは「表示する」をOFFにして残しておくのがおすすめです。

## 洋食ツバキ亭のメニューを変更する

1. microCMSにログインする
2. つばき亭のサービスを開く
3. 「メニュー」APIを開く
4. 変更したい商品を探す
5. 必要な項目だけ変更する
   - 品名
   - 説明
   - 金額
   - 金額表示
   - 写真
   - 表示する
6. 保存する
7. ホームページのメニューページを開き直して確認する

洋食ツバキ亭は、ランチ・ディナー・コースなどを同じ「メニュー」APIの中で管理します。

「メニュー区分」や「セクション名」は基本的に変更しないでください。

## ツバキ&デリのメニューを変更する

ツバキ&デリには、あらかじめ20個のメニュー枠を用意します。

最初は次のような仮表示です。

```text
coming soon...
￥...
```

実メニューに差し替える時は、空いている枠を開いて次の項目を入れます。

- 品名
- 金額 または 金額表示
- 写真
- 表示する

不要な枠は「表示する」をOFFにしてください。

20個より多くメニューを出したくなった場合は、Yoshi側に追加作業を依頼してください。

## 新しいメニューを追加したい時

### ツバキ&デリの場合

まず、未使用の「coming soon...」枠を使ってください。

1. 未使用の枠を開く
2. 品名を変更する
3. 金額または金額表示を入れる
4. 写真を入れる
5. 「表示する」をONにする
6. 保存する

### 洋食ツバキ亭の場合

洋食ツバキ亭のメニュー追加は、表示場所や並び順に関係します。

追加したい場合は、Yoshi側に依頼してください。

## メニューを消したい時

削除ではなく「表示する」をOFFにしてください。

削除すると、あとで復活させる時に再入力が必要になります。

## 保存したのに反映されない時

次の順番で確認してください。

1. microCMSで保存できているか確認する
2. ホームページを再読み込みする
3. スマホの場合は、ブラウザを一度閉じて開き直す
4. それでも変わらない場合は、数分待ってから再読み込みする
5. まだ変わらない場合は、Yoshi側に連絡する

## 金額が思った通りに出ない時

「金額表示」が入っている場合は、「金額」よりも「金額表示」が優先されます。

普通の金額表示に戻したい場合は、「金額表示」を空欄にしてください。

## 写真が出ない時

次を確認してください。

- 写真のアップロードが完了しているか
- 保存できているか
- 写真欄に古い画像と新しい画像が混ざっていないか
- ホームページを再読み込みしたか

それでも出ない場合は、Yoshi側に連絡してください。

## 絶対にやらないこと

次の操作は、基本的にしないでください。

- メニュー項目を削除する
- 店舗を変更する
- メニュー区分を変更する
- 並び順を大きく変更する
- APIキーを他人に送る
- microCMSの設定画面を変更する

## 初期設定後にパスワードを変更する

今回は仮運用として、初期設定のためにログイン情報をYoshi側へ共有する場合があります。

初期設定が終わったら、つばき亭さん側でパスワードを変更してください。

その後、Yoshi側が普段ログインする必要は基本的にありません。

## 画像つきマニュアルでQRにするリンク

画像つきPDF版では、次のリンクをQRコードにします。

- microCMSログイン画面
- 洋食ツバキ亭 メニューページ
- ツバキ&デリ メニューページ
- 必要であれば、Yoshiへの連絡先

正式なURLは、初期設定とドメイン切替の状況に合わせて最終版で入れます。
