# つばき亭 メニュー更新マニュアル（文字だけ版）

このマニュアルは、microCMSからホームページのメニューを変更するための説明です。

対象は次の2店舗です。

- 洋食ツバキ亭
- ツバキ&デリ

microCMSで変更した内容は、ホームページのPC表示とスマホ表示の両方に反映されます。PC用とスマホ用を別々に直す必要はありません。

## 使うリンク

- microCMS管理画面: https://tsubakitei-hp.microcms.io/
- 洋食ツバキ亭 メニューページ: https://toraikura.github.io/tsubakitei-hp/menu/
- 洋食ツバキ亭 メニューページ 変更後アドレス: https://tsubakitei.tokyo/menu/
- ツバキ&デリ メニューページ: https://toraikura.github.io/tsubakitei-hp/deli/menu/
- ツバキ&デリ メニューページ 変更後アドレス: https://tsubakitei.tokyo/deli/menu/

ドメイン切替後は、ホームページ側のURLだけ正式ドメインに変わる予定です。microCMSの管理画面はそのまま使います。

## 最初に知っておくこと

microCMSには、店舗ごとに別のメニュー管理画面があります。

- 洋食ツバキ亭のメニューを変える時: `ツバキ亭メニュー`
- ツバキ&デリのメニューを変える時: `デリメニュー`

別店舗のメニューに間違えて入ると、違うページを直してしまいます。まず左側のメニュー名を確認してください。

## 普段触ってよい項目

普段変更してよいのは、基本的に次の項目です。

- 表示する
- 品名
- 金額表示
- 写真

洋食ツバキ亭の場合だけ、必要に応じて次も使います。

- メニュー区分
- セクション名
- セクション説明

ただし、`メニュー区分`と`セクション名`は必ずプルダウンから選んでください。手入力欄が出てきても、自由に文字を打つ運用にはしません。入力ミスがあるとホームページの表示場所が崩れる原因になります。

## 普段触らない項目

次の項目は、基本的に触らないでください。

- 並び順
- 既存写真パス
- API設定
- カスタムフィールド設定
- APIキー

`並び順`を変えると表示順が変わります。小さい数字ほど上に出ます。必要な時だけ麻生側で調整します。

`既存写真パス`は、最初からホームページ内に入っている写真を表示するための欄です。新しい写真へ差し替える場合は、ここではなく`写真`欄を使ってください。

## 金額の入れ方

金額は`金額表示`に、ホームページへ出したい文字をそのまま入力します。

例:

```text
￥1,200
```

複数価格の場合も、そのまま入力します。

例:

```text
S ￥650 / L ￥750
```

価格未定や準備中の場合も、そのまま入力できます。

例:

```text
￥...
```

数字だけの金額欄は使いません。迷ったら`金額表示`だけ直せばOKです。

## 写真の変え方

1. microCMSで変更したい商品を開く
2. `写真`の欄を探す
3. 新しい写真をアップロードする
4. 保存する
5. ホームページを再読み込みして確認する

`写真`欄に画像を入れると、その画像が優先してホームページに表示されます。

写真が入っていない場合は、`既存写真パス`に入っている元写真、または空の枠が表示されます。

## 表示・非表示の切り替え

一時的にメニューを隠したい時は、`表示する`をOFFにします。

復活させたい時は、`表示する`をONにします。

削除は基本的にしないでください。削除すると、あとで戻す時に再入力が必要になります。

## ツバキ&デリのメニューを変更する

ツバキ&デリは、`デリメニュー`で管理します。

デリには、あらかじめ複数のメニュー枠を用意しています。未使用の枠は次のような表示です。

```text
coming soon...
￥...
```

新しい商品を出す時は、未使用の枠を上書きします。

1. microCMSにログインする
2. 左側から`デリメニュー`を開く
3. `coming soon...`の未使用枠を開く
4. `品名`を入力する
5. `金額表示`を入力する
6. 必要なら`写真`をアップロードする
7. `表示する`をONにする
8. 保存する
9. デリメニューページを再読み込みして確認する

使わない枠は、削除せずに`表示する`をOFFにしてください。

## 洋食ツバキ亭のメニューを変更する

洋食ツバキ亭は、`ツバキ亭メニュー`で管理します。

ランチ、ディナー、コースは同じ`ツバキ亭メニュー`の中に入っています。

1. microCMSにログインする
2. 左側から`ツバキ亭メニュー`を開く
3. 変更したい商品を開く
4. 必要な項目だけ変更する
   - `品名`
   - `金額表示`
   - `写真`
   - `表示する`
5. 保存する
6. 洋食ツバキ亭メニューページを再読み込みして確認する

`メニュー区分`と`セクション名`は、商品の表示場所を決める項目です。基本的には変更しません。

変更が必要な場合は、必ずプルダウンから選んでください。

## 洋食ツバキ亭で新しい商品を追加する時

洋食ツバキ亭で新しい商品を追加する場合は、表示場所を選ぶ必要があります。

1. `ツバキ亭メニュー`を開く
2. `追加`を押す
3. `表示する`をONにする
4. `メニュー区分`をプルダウンから選ぶ
5. `セクション名`をプルダウンから選ぶ
6. `品名`を入力する
7. `金額表示`を入力する
8. 必要なら`写真`をアップロードする
9. `並び順`を入力する
10. 保存する
11. ホームページで表示場所を確認する

選択肢の目安:

```text
Lunch  = ランチ
Dinner = ディナー
Course = コース
```

`並び順`は小さい数字ほど上に表示されます。基本は10、20、30のように10ずつ空けて入れます。

追加に迷う場合は、麻生側に依頼してください。

## コース料理を直す時

コース料理は、通常商品より少し注意が必要です。

`コース料理`全体の説明は、`セクション説明`に入ります。

各コースごとの説明や価格は、商品ごとの欄に入ります。

コースの見た目が崩れた場合は、無理に直さず麻生側に連絡してください。コースは表示崩れが起きやすい場所です。

## 保存したのに反映されない時

次の順番で確認してください。

1. microCMSで保存できているか確認する
2. ホームページを再読み込みする
3. スマホの場合は、ブラウザを一度閉じて開き直す
4. 数分待ってから、もう一度再読み込みする
5. それでも変わらない場合は、麻生側に連絡する

変更はすぐ反映されることが多いですが、ブラウザ側に古い表示が残ることがあります。

## 写真が出ない時

次を確認してください。

- 写真のアップロードが完了しているか
- 保存できているか
- ホームページを再読み込みしたか
- `表示する`がONになっているか

それでも出ない場合は、麻生側に連絡してください。

## よくあるミス

### 別店舗のメニューを開いている

デリを直したい時は`デリメニュー`、洋食ツバキ亭を直したい時は`ツバキ亭メニュー`を開いてください。

### 保存せずに画面を閉じている

入力後は必ず保存してください。

### `表示する`がOFFになっている

品名や金額を入れても、`表示する`がOFFだとホームページには出ません。

### プルダウンではなく手入力している

洋食ツバキ亭の`メニュー区分`と`セクション名`は、必ず選択式で入れてください。

### 金額の文字が半角・全角でバラバラ

見た目を揃えるため、基本は次の形にします。

```text
￥1,200
```

## 絶対にやらないこと

次の操作はしないでください。

- API設定を変更する
- カスタムフィールド設定を変更する
- APIキーをコピーして他人に送る
- APIキーを作り直す
- メニュー項目を大量に削除する
- `既存写真パス`をまとめて消す
- `ツバキ亭メニュー`と`デリメニュー`を混ぜて編集する

設定画面を変更すると、ホームページに表示されなくなることがあります。

## 困った時に麻生側へ送るもの

困った時は、次の情報を送ると確認が早いです。

- どちらの店舗か
- どの商品か
- 何を変更したか
- microCMSの画面スクリーンショット
- ホームページの画面スクリーンショット

例:

```text
デリメニューの「洋食屋のビーフストロガノフ」の写真を変えたけど、ホームページに反映されません。
```
